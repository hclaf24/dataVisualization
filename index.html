Good Data Visualization

I think this is a great data visualization example because it has consistent scaling and effective color usage. The color legend is displayed prominently at the top of the figure. Additionally, both axes have consistent scaling and are labeled.

Bad Data Visualization

This data visualization is not very helpful. On the Y axis we have abbreviations for cities but no label to help the viewer come to that conclusion. The legend is placed in the bottom right corner, which for the typical english reader, is the last place one looks. The colors I believe represent soccer positions but again there is little labeling, just a letter. It is hard to use this graph as an image, however, it is an interactive graph which gives more information when you hover over the different sections.
